# 🚀 First-Year Computer Science Journey (C++)

Welcome to my educational repository! This space serves as a clean, curated digital portfolio documenting my academic progress and coding journey during my **First Year of Computer Science Engineering** at **Manara University** (2025-2026).

## 📌 Repository Strategy
Unlike cluttered repositories, this profile follows a **strict no-redundancy rule**. Every single file here has been carefully filtered to represent a unique programming concept, core algorithm, or memory management technique, making it highly scannable and professional.

## 🛠️ Core Concepts & Algorithms Covered

### 🔹 Fundamentals & Logic
* **Control Flow:** Complex conditional logic (`if-else`, `switch-case`) and standard loops (`for`, `while`, `do-while`).
* **Advanced Functions:** Function overloading, function templates, recursion limits, and understanding variable scopes (Global vs. Local vs. Static).

### 🔹 Data Structures & Array Manipulation
* **1D & 2D Arrays:** Matrix arithmetic, conditional printing, row/column summation, and dynamic memory allocation.
* **Strings:** Character array manipulation and processing raw text inputs.

### 🔹 Advanced Pointers & Memory Management
* **Pointer Arithmetic:** Navigating arrays using different pointer notations and offsets.
* **Reference vs. Pointer:** Mastering pass-by-value, pass-by-reference, and pass-by-pointer concepts.

### 🔹 Classic Computer Science Algorithms
* **Sorting:** Bubble Sort algorithm implementation.
* **Searching:** Linear Search and Binary Search algorithms.
* **Recursion:** Classic problems including Factorial logic, Fibonacci series, Pascal Triangle simulations, and the Tower of Hanoi.
* **Mini-Projects:** Time-seeded random number generators (RNG), number guessing games, and a full Card Shuffling and Dealing simulator.

## 💻 Tech Stack & Tools
* **Language:** C++ (ISO Standard)
* **IDE & Environment:** Code::Blocks / Visual Studio Code
* **Version Control:** Git & GitHub Desktop (Activated via GitHub Student Developer Pack)
